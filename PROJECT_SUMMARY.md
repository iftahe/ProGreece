# סיכום התקדמות פרויקט ProGreece

מסמך זה מרכז את כל העבודה שבוצעה בפרויקט, כולל שדרוגים אחרונים במודל הנתונים וניהול הפרויקטים.

## 1. סקירה כללית (Overview)
המערכת היא אפליקציית Web לניהול תזרים מזומנים (Cash Flow) וניהול תקציב פרויקטלי עבור "Greece Project". המערכת תומכת כעת בריבוי פרויקטים, ניהול תקציב (Budget vs Actual) וניהול תנועות מתקדם.

---

## 2. ארכיטקטורה וטכנולוגיות
*   **Backend**: Python (FastAPI), SQLAlchemy, SQLite/PostgreSQL.
*   **Frontend**: React, Vite, Tailwind CSS.
*   **Infrastructure**: מוכן לפריסה ב-Render.com (`render.yaml`).

---

## 3. מודלים ונתונים (Database Schema Updates)
בוצע שדרוג משמעותי למודל הנתונים לתמיכה בריבוי פרויקטים ותקציב:

*   **Projects**: טבלה משודרגת הכוללת סטטוס (Active/Archived), תקציב כולל (`total_budget`) ושדות לניהול שוטף.
*   **BudgetCategory** (חדש): ניהול עץ תקציב (סעיפים ותתי-סעיפים). מקושר לפרויקט.
*   **Transactions**:
    *   קישור לתקציב: שדה `budget_item_id` המקשר כל תנועה לסעיף תקציבי.
    *   תמיכה בסטטוס תכנון מול ביצוע.

---

## 4. פיצ'רים מרכזיים

### א. ניהול פרויקטים (Project Management) - **חדש**
*   **מסך פרויקטים (`Projects.jsx`)**: צפייה ברשימת הפרויקטים, יצירת פרויקט חדש, עריכת פרטים (שם, תקציב, סטטוס) וניהול סטטוס פרויקט.
*   **Backend API**: הוספו Endpoints ליצירה (`POST`), עדכון (`PUT`) ושליפת פרויקטים.

### ב. ניהול תנועות ותקציב
*   **שיוך לתקציב**: בעתיד הקרוב, מסך התנועות יאפשר בחירת סעיף תקציבי לכל הוצאה/הכנסה.
*   **מעקב תקציב**: התשתית מוכנה להשוואת מתוכנן (Budget) מול ביצוע (Actual) ברמת הסעיף הבודד.

### ג. דאשבורד ותזרים
*   **Cash Flow**: תצוגה גרפית וטבלאית של תזרים המזומנים הצפוי והבפועל.

### ד. יבוא נתונים
*   סקריפטים לייבוא אוטומטי של תנועות בנק ותוכניות (Plans).
